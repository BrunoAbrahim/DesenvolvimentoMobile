# Projeto da Calculadora


## Requisitos Funcionais:
- RF01: A aplicação deve permitir a inserção de números de 0 a 9 por meio de botões na interface.
- RF02: Os números digitados devem ser exibidos no display principal.
- RF03: A aplicação deve realizar as operações de adição (+), subtração (-), multiplicação (×), divisão (÷) e porcentagem (%).
- RF04: Ao pressionar o botão "=", a aplicação deve calcular e exibir o resultado da operação.
- RF05: Ao pressionar o botão "AC", a aplicação deve limpar todos os valores e reiniciar o estado da calculadora.
- RF06: A aplicação deve permitir alternar o sinal do número atual utilizando o botão "+/-".
- RF07: A aplicação deve permitir a inserção de números decimais utilizando o ponto (.).
- RF08: A aplicação não deve permitir mais de um ponto decimal no mesmo número.
- RF09: A aplicação deve exibir a expressão matemática digitada em um campo secundário acima do resultado.
- RF10: A aplicação deve impedir divisão por zero.
- RF11: A aplicação deve impedir operadores consecutivos inválidos.
- RF12: A aplicação deve impedir o cálculo de operações incompletas.

## Não Funcionais
- RNF01: A aplicação deve ser desenvolvida utilizando React Native.
- RNF02: A aplicação deve funcionar em dispositivos Android e iOS.
- RNF03: A interface deve ser responsiva e seguir o layout definido no protótipo.
- RNF04: A aplicação deve responder aos comandos do usuário sem atrasos perceptíveis.
- RNF05: Os botões devem ter tamanho adequado para interação em dispositivos móveis.
- RNF06: O código deve ser organizado em componentes reutilizáveis.
- RNF07: A lógica de cálculo deve ser separada da camada de interface.
- RNF08: O código deve ser estruturado de forma a facilitar manutenção e futuras melhorias.
